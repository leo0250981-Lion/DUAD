\# Student Management System



This project is a command-line interface application that allows the user to manage student academic information such as grades, sections, averages, and performance.



Features:

\- Add new students

\- Validate name, section, and grades

\- Display all students

\- Top 3 students by average

\- Global average score

\- Export and import data via CSV

\- Delete students

\- Show failing students
