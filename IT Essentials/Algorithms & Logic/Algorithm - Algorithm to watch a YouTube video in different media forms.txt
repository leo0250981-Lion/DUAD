ALGORITHM TO WATCH A VIDEO ON YOUTUBE (PC, MOBILE, TABLET, AND SMART TV)
A. USING A PC

1- Start

2- Turn on the computer

3- If it is not on:
a. Turn it on

4- If it is on:
a. Wait for the system to load and display the interface and files

5- If it has not finished loading:
a. Wait until the system is ready

6- If it is ready:
a. Click on the web browser

7- If the browser is not open:
a. Wait for it to load

8- Type "www.youtube.com
" in the search bar and press Enter

9- Wait for YouTube to load

10- Search for the video by typing the name in the search bar

11- Select the desired video

12- End (PC)

B. USING A MOBILE DEVICE (ANDROID OR IPHONE)

1- Start

2- Check if the phone is turned on

3- If it is not on:
a. Press and hold the power button
b. Turn on the device

4- If it is already on:
a. Unlock the screen

5- Look for the YouTube app

6- If it does not appear:
a. Swipe up to open the app menu
b. Search YouTube in the list

7- Tap the YouTube icon

8- Wait for the app to open

9- Tap the search bar at the top

10- Type the name of the video

11- Select the video from the list

12- End (Mobile)

C. USING A TABLET (ANDROID OR IPAD)

1- Start

2- Turn on the tablet

3- If it is not on:
a. Press and hold the power button

4- If it is on:
a. Unlock the screen

5- Locate and open the YouTube app

6- If the app does not load:
a. Wait a few seconds
b. Try opening it again

7- Tap the search bar

8- Type the name of the video

9- Select the video

10- End (Tablet)

D. USING A SMART TV

1- Start

2- Turn on the TV

3- If it is not on:
a. Turn it on with the remote control

4- If the Smart TV is already on:
a. Wait for the main menu to load

5- Find the YouTube app in the menu

6- If it does not appear:
a. Go to “Apps” or “Smart Hub”
b. Search for YouTube

7- Select YouTube using the remote

8- Wait for it to load

9- Go to the search bar (magnifying glass icon)

10- Enter the name of the video using the on-screen keyboard
or
Use voice control if the TV has a microphone (e.g., “Search relaxing music”)

11- Select the video

12- End (Smart TV)
